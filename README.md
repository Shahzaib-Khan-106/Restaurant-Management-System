# Restaurant Management System

A Flutter-based application for managing restaurant operations with **role-based dashboards**.

---

## 🚀 Features
- **Admin** → Access to all modules
- **Kitchen Staff** → Inventory & Recipe Management
- **Procurement Officer** → Orders & Suppliers
- **Accountant** → Analytics Dashboard

---

## 🛠️ Tech Stack
- **Frontend:** Flutter, Dart
- **Backend (Planned):** Firebase Auth, Firestore
- **Platforms:** Android, iOS, Web, Windows Desktop
